# ETF 모멘텀 계산기 (복제본)

원본: https://etf-v6-c3-617871724184.us-west1.run.app/

Google AI Studio로 빌드되어 Cloud Run에 배포된 앱을 페이지에 내장된 소스(importmap의 base64 모듈)를 디코딩해 재구성한 정적 웹앱입니다. 소유자가 아니라 원본 AI Studio 프로젝트에는 접근할 수 없어, 배포된 페이지에서 소스를 추출하는 방식으로 복제했습니다.

## 포함된 기능
- 공공데이터포털(`apis.data.go.kr`) ETF 시세 API를 브라우저에서 직접 호출해 기간별 수익률/모멘텀 점수 계산
- 가중치·최소 거래량·최소 거래대금·표시 순위·순위 비교 기간 설정
- 보조 지표(Stochastic, RSI, MACD) 필터링
- 종목별 상세 차트 (Chart.js: 가격/거래량, 모멘텀, Stochastic, RSI, MACD)
- 섹터 분류·범례, 신규 상장 ETF 표시
- 데이터 캐시 불러오기/생성/업데이트/다운로드, CSV 내보내기

## 제외된 기능 (의도적)
원본은 Gemini API(`@google/genai`)를 썼는데, 실제 배포본에서는 Cloud Run 서버가 `/api-proxy`로 API 키를 주입하는 백엔드가 있어야 동작합니다. 이 정적 복제본에는 그 백엔드가 없어 아래 두 기능은 비활성화(클릭 시 안내 메시지만 표시)되어 있습니다.
- 종목 상세 모달의 "최신 정보 분석하기" (AI 기반 종목 설명/보유종목/뉴스·영상 검색어)
- "결과 분석하기" (AI 기반 상승/하락 섹터 종합 분석) 및 그 결과를 내보내는 NotebookLM 콘텐츠 내보내기

필요하다면 별도 안내에 따라 이 기능들을 되살릴 수 있습니다. 자신의 Gemini API 키로 직접 `@google/genai`를 클라이언트에서 호출하거나(키가 브라우저에 노출됨), 간단한 Node/Express 프록시 서버를 앞단에 두는 두 가지 방법이 있습니다.

## 로컬 실행
정적 파일이라 아무 정적 서버로 실행하면 됩니다.

```
python -m http.server 8000
```

이후 브라우저에서 `http://localhost:8000` 접속. "공공데이터포털 서비스 키"는 본인이 [공공데이터포털](https://www.data.go.kr/)에서 발급받은 서비스 키를 직접 입력합니다(브라우저에만 저장, 서버로 전송되지 않음).

## 배포
정적 호스팅(GitHub Pages, Firebase Hosting, Cloud Run의 정적 컨테이너, Netlify/Vercel 등) 어디든 `index.html`과 `app.js`만 올리면 동작합니다. 백엔드가 필요 없습니다.
