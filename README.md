# 🔭 ETF MomentoScope

공공데이터포털 API로 기간별 ETF 수익률과 모멘텀 점수를 계산하는 정적 웹앱입니다.

**라이브**: https://jgjeong730.github.io/ETF_Momentum/

원본은 Google AI Studio로 빌드되어 Cloud Run에 배포된 데모(`etf-v6-c3-*.run.app`)였습니다. 소유자가 아니라 원본 프로젝트에는 접근할 수 없어, 페이지에 내장된 소스(importmap의 base64 모듈)를 디코딩해 재구성한 뒤 GitHub Pages로 새로 배포했습니다.

## 빌드 워크플로우

|---|---|---|

## 포함된 기능
- 공공데이터포털(`apis.data.go.kr`) ETF 시세 API를 브라우저에서 직접 호출해 기간별 수익률/모멘텀 점수 계산 (1주/2주/1개월/3개월/6개월/12개월 가중치, 합계 100 검증)
- 최소 거래량·최소 거래대금·표시 순위·순위 비교 기간 설정
- 종목찾기 (직전 계산 결과 내 검색) & 관심종목 (localStorage 저장, 순서 변경 가능)
- 종목별 상세 차트 (Chart.js: 가격/거래량, 일일 모멘텀, 스토캐스틱, RSI, MACD) — 1주~1년 기간 선택
- 운용 성과 요약 (기간별 수익률, 연율화 변동성, 누적수익률 그래프)
- 섹터 분류·범례, 신규 상장 ETF 표시, CSV 내보내기

## 제외된 기능 (의도적)
원본은 Gemini API(`@google/genai`)를 썼는데, Cloud Run 서버가 `/api-proxy`로 API 키를 주입하는 백엔드가 있어야 동작합니다. 이 정적 사이트에는 그 백엔드가 없어 AI 기반 종목 설명·보유종목 조회, 시장 종합 분석 기능은 제외했습니다. 필요하면 본인 Gemini API 키로 직접 호출하거나(키가 브라우저에 노출됨) 간단한 서버리스 프록시를 앞단에 두는 방식으로 되살릴 수 있습니다.

## 로컬 실행
정적 파일이라 아무 정적 서버로 실행하면 됩니다.

```
python -m http.server 8000
```

이후 브라우저에서 `http://localhost:8000` 접속. "공공데이터포털 서비스 키"는 본인이 [공공데이터포털](https://www.data.go.kr/)에서 발급받은 서비스 키를 직접 입력합니다(브라우저에만 저장, 서버로 전송되지 않음).

## 배포
정적 호스팅(GitHub Pages, Firebase Hosting, Netlify/Vercel 등) 어디든 `index.html`과 `app.js`만 올리면 동작합니다. 백엔드가 필요 없습니다.
